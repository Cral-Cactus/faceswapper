# FaceSwapper

FaceSwapper is a versatile tool designed for advanced facial processing tasks including face swapping, enhancement, upscaling, and more. It utilizes cutting-edge deep learning models to achieve high-quality results in various scenarios.

## Features

- **Face Swapping:**
  - Swap faces between images and videos using 5 different models.
  - Swap faces between two images.

- **Face Enhancement:**
  - Enhance facial features using 7 different models.
  - Improve facial details and quality.

- **Image and Video Enhancement:**
  - Enhance and upscale images and videos using 3 different models.
  - Improve overall quality and resolution.

- **Lip Sync:**
  - Sync facial movements and expressions with audio to generate realistic talking head videos.

- **Multi-Model Face Detection:**
  - Robust face detection using multiple models for accurate identification and tracking.

- **Occlusion and Region Masking:**
  - Mask specific regions such as eyes, mouth, and nose to prevent distortion or unwanted changes during editing.

## Models Available

FaceSwapper utilizes state-of-the-art models for each functionality:

- **Face Swapping Models:** Model A, Model B, Model C, Model D, Model E
- **Face Enhancement Models:** Model X, Model Y, Model Z, Model 1, Model 2, Model 3, Model 4
- **Image/Video Enhancement Models:** Model P, Model Q, Model R
- **Lip Sync Model:** Model L
- **Face Detection Models:** Model FD1, Model FD2
- **Masking Models:** Occlusion Mask, Region Mask (Eyes, Mouth, Nose)
